Project Name: Personalized Data API

Description: An API service that lets users interact with an eCommerce server.

Getting Started:

Prerequisites:
        Java 17 (or higher),
        Maven 3.8.6

Clone the Project:

    git clone https://github.com/uvin6667/personalized-data-api.git

Build the Project:

    mvn clean package 

Run the Application:

    mvn spring-boot:run
